# 📸 SwaaFotoes Studio By Afgan

SwaaFotoes Studio adalah web-based self photo studio yang gue bikin buat kasih lo pengalaman foto ala booth premium. Aplikasi ini simpel, ringan, dan hasilnya bisa langsung lo download dengan filter yang udah nempel permanen.

---

### 🔥 Fitur Unggulan

- **Permanent Filter Baking**: Filter (Vivid, Noir, Pink) gak cuma buat preview, tapi bakal kebakar (baked) langsung ke hasil foto pas lo download.
- **Baking Animation**: Pas lo simpan foto, ada progress bar real-time yang nunjukin proses "memasak" foto kualitas tinggi.
- **Studio Music**: Ada player musik buat nemenin sesi foto lo biar gak sepi.
- **Flexible Layout**: Bisa pilih mau mode **STRIP** (1x4) atau **GRID** (2x2).
- **Auto-Cycle Stickers**: Stiker `pindah1` sampe `pindah3` bakal ganti-ganti otomatis biar foto lo lebih hidup.

---

### 📂 Persiapan Aset

Biar gak ada error pas jalanin kodenya, pastiin lo punya file aset ini di folder yang sama:

| Kategori | Nama File |
| --- | --- |
| **Utama** | `index.html` |
| **Audio** | `musik.mp3` |
| **Stiker** | `pindah1.png`, `pindah2.png`, `pindah3.png` |
| **Ikon** | `adjust.png`, `flip.png`, `retry.png`, `play.png`, `pause.png` |

---

### 🚀 Cara Pake

1. **Siapin File**: Download semua aset dan taruh dalam satu folder bareng `index.html`.
2. **Buka Browser**: Klik kanan di `index.html` terus buka pake Chrome atau Safari.
3. **Izin Kamera**: Pas muncul notif, klik **Allow** buat kamera.
4. **Foto & Edit**: Pilih filter, ambil 4 pose, pilih warna background kartu lo.
5. **Simpan**: Klik **Download High-Res**, tunggu proses baking kelar, dan foto langsung masuk ke galeri lo.
